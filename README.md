# Приложение по поиску фильмов

#### Возможности: ### 
- Cортировка: имя, год выпуска, жанр, рейтинг.
- Детальная информация по фильму, похожие фильмы.
- Сохранение фильма в избранное.
  
 ### Api: ###
 - https://kinopoisk.dev/
  
 ### Технологии ###
 - MVVM.
 - Coroutines. 
 - Flow.
 - Navigation. 
 - DAO Room. 
 - Moshi. 
 - Retrofit/OkHttp2. 
 - Dagger Hilt.
 - Material.
